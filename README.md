# Setup

```bash
# grant permissions
$ chmod 777 index.js
# create local env file from example
$ cp .env.example .env
# create symlink
$ npm link
```

# Run

````bash
# run command
$ ./index.js <cmd>
```# base-node-cli
````

# Test 

````bash
# run command
$ ./index.js hello -p Robbie 
```# base-node-cli
````
